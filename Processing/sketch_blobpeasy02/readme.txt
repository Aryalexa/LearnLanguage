esto es para
leer datos de la imagen (los colores pixel a pixel)

y hacer el blob detection

el archivo con los valores debe tener los enteros que forman la imagen uno detras de otro separados por saltos de linea. Los colores tienen que estar representados por enteros y se interpretar de la forma 
0xaaRRGGBB
(alpha,red,green,blue)
4 bytes = 32 bits
